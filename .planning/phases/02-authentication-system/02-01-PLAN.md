   - hashed_password: String(255), not nullable
   - full_name: String(255), nullable
   - is_active: Boolean, default=True
   - created_at: DateTime, default=datetime.utcnow
   - updated_at: DateTime, default=datetime.utcnow, onupdate=datetime.utcnow

6. Add __repr__ method for debugging

7. Import Base from backend.app.database at module level

DO NOT: Store plain text passwords, use hashed_password field only
DO NOT: Use simple string IDs - use UUID for security
DO NOT: Include password field in User model - only hashed_password
